Plone Lock Manager

    This product allows you to manage Web DAV locks in a Plone site through the Plone interface. It adds in a control panel option for searching for locks. It also adds in a link on the folder contents page as a document action that will give you a list of locks.

Author:

    Sidnei Da Silva, Enfold Systems LLC

License:

    GPL

Installation:

    Standard Plone way. Copy this containing directory (PloneLockManager) into the Products directory of your Plone instance. Restart. Go to plone setup > add remove product and install PloneLockManager.
