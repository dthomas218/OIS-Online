This product provides some of the server side components for Plone Desktop. It contains any custom skins and code, plus some patches for fixing up things in Plone that shouldn't happen the way they do.

Dependencies:

    PropertySets and CMFPropertySets

    Both of these are included in the install of Plone Desktop.

Installation:

    As usual unpack into your Products directory, restart Zope.

    Go to Add/Remove Products in plone setup and install this product.
